# TASK 2: Reflection Notes

To set myself up for success over the next four years, I’ve chosen to focus on GitHub, LinkedIn, and Stack Overflow. I see GitHub as my digital workshop; it’s where I’ll host my code, practice version control, and eventually contribute to open-source projects to show what I can actually build. 

LinkedIn is my professional home base, which I’ll use to connect with mentors, follow industry trends, and build a personal brand that goes beyond just a resume. 

Stack Overflow is my go-to for problem-solving; it’s an incredible place to find answers to tough coding bugs and, eventually, a place where I can give back by helping others.
During my first two years, I plan to use GitHub mainly for university assignments and Stack Overflow to help me get through the steep learning curve of new languages. As I move into my third and fourth years, I’ll shift my focus toward my professional presence. I’ll start sharing my milestones on LinkedIn to attract internship opportunities and refine my GitHub profile to act as a portfolio for recruiters. By graduation, I hope to have a solid reputation across all three platforms, transitioning from a student who asks questions to a professional who provides solutions.
